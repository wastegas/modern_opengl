Modern openGL
=============
Basics of modern openGL

Prerequisites
-------------
- [OpenGL](https://www.khronos.org/opengl/wiki/Getting_Started/)
- [GLFW](https://glfw.org)
- [GLAD](https://glad.dav1d.de)
- [stb](https://github.com/nothings/stb#stb_libs)

Build
-----
```
mkdir build
cd build
cmake ..
make
```
